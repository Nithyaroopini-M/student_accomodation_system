
# 🏠 Accommodation Finder & Booking System

A full-stack web application that helps students and working professionals find and book hostels or PG accommodations. The system allows users to search for accommodations based on their preferences, while hostel/PG owners can manage their listings and booking requests through a dedicated dashboard.

---

## Features

### User
- Register and Login
- Search accommodations
- Filter by location, budget, and amenities
- View accommodation details
- Send booking requests
- View booking status

### Owner
- Register and Login
- Add accommodation listings
- Update or delete listings
- Manage room availability
- Accept or reject booking requests
